# Azure Cloud Resume

My implementation of the [Cloud Resume Challenge](https://cloudresumechallenge.dev) on Azure.

**Live site:** _coming soon_

## Architecture
- Frontend: Azure Static Web Apps (HTML/CSS/JS)
- API: Azure Functions (Python 3.11)
- Database: Azure Cosmos DB (NoSQL, serverless)
- IaC: Bicep
- CI/CD: GitHub Actions
